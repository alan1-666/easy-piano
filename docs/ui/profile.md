> 版本: v1.0.0 | 更新日期: 2026-04-18

## 1. 页面概述

个人中心展示用户的学习成就、练习数据和个人设置入口。是用户了解自身进度和调整偏好的核心页面。

## 6. 交互说明

| 操作 | 行为 |
|------|------|
| 点击头像 | 弹出选项: 拍照 / 相册选择 / 查看大图 |
| 点击统计卡片 | 跳转到对应详情 (如练习历史列表) |
| 点击热力图某天 | 弹出 Tooltip "1月15日，练习32分钟" |
| 点击已获得成就 | 弹出成就详情弹窗 |
| 点击未获得成就 | 弹出解锁条件和当前进度 |
| 点击 ⚙ | 进入设置页 |
| 下拉刷新 | 同步最新数据 |

## 7. 边界情况

| 场景 | 处理 |
|------|------|
| 未登录/游客 | 显示 "登录后查看完整数据" + 登录按钮 |
| 新用户 (无数据) | 统计显示 0，热力图空白，成就全部锁定 |
| 头像加载失败 | 显示默认头像 (用户名首字母) |
| 设置项保存 | 实时保存，无需确认按钮 |
| 退出登录 | 弹窗确认 "确定要退出登录吗？" |
| 清除缓存 | 弹窗确认 + 进度指示器 + 完成 Toast |
