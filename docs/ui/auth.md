# 登录注册页

> 用户认证流程  
> 版本: v1.0.0 | 更新日期: 2026-04-18

## 1. 页面概述

提供邮箱/手机号注册、Apple ID 登录和微信登录三种认证方式。支持游客模式跳过登录。

## 6. 表单验证

### 6.1 验证规则

| 字段 | 规则 | 错误提示 |
|------|------|---------|
| 邮箱 | 合法邮箱格式 | "请输入有效的邮箱地址" |
| 手机号 | 11位数字，1开头 | "请输入有效的手机号" |
| 用户名 | 2-20字符，不含特殊字符 | "用户名需 2-20 个字符" |
| 密码 | ≥6位，包含字母和数字 | "密码至少6位，需包含字母和数字" |
| 确认密码 | 与密码一致 | "两次密码输入不一致" |
| 条款 | 必须勾选 | "请阅读并同意用户协议" |

### 6.2 验证时机

| 时机 | 行为 |
|------|------|
| 输入时 | 实时验证格式 (300ms 防抖)，错误状态红色边框 |
| 失焦时 | 完整验证，显示错误提示文字 |
| 提交时 | 全部验证，定位到第一个错误字段 |

## 7. 第三方登录流程

### 7.1 Apple ID 登录

```
点击 "通过 Apple 登录"
    │
    ▼
系统 Apple ID 授权弹窗 (系统控件)
    │
    ├── 授权成功 → 检查后端是否有账号
    │                 │
    │                 ├── 有 → 直接登录 → 首页
    │                 │
    │                 └── 无 → 自动创建账号 → 首页
    │
    └── 取消/失败 → 返回登录页，Toast "登录取消"
```

### 7.2 微信登录

```
点击 "微信登录"
    │
    ▼
检查是否安装微信
    │
    ├── 未安装 → Toast "请先安装微信"
    │
    └── 已安装 → 跳转微信授权
                    │
                    ├── 授权成功 → 获取 openid → 登录/注册 → 首页
                    │
                    └── 取消 → 返回登录页
```

## 8. 交互说明

| 操作 | 行为 |
|------|------|
| 点击"立即注册" | 跳转注册页 (push 导航) |
| 点击"立即登录" | 跳转登录页 (push 导航) |
| 点击"跳过" | 以游客模式进入首页 (功能受限) |
| 点击"忘记密码" | 跳转忘记密码页 |
| 键盘"完成" | 提交表单 |
| 键盘"下一项" | 焦点移到下一个输入框 |

## 9. 边界情况

| 场景 | 处理 |
|------|------|
| 网络超时 | 登录按钮恢复可点击 + Toast "网络超时，请重试" |
| 邮箱已注册 | 提示 "该邮箱已注册，请直接登录" + 登录链接 |
| 密码错误 | 提示 "邮箱或密码错误" (不区分) + 错误次数计数 |
| 连续 5 次错误 | 锁定登录 60 秒 + 倒计时 |
| 游客升级 | 游客模式下，在需要账号功能处提示登录，数据自动绑定 |
| 键盘遮挡 | 页面自动上推，确保当前输入框可见 |
| App 被系统终止 | 已输入内容不保留 (安全考虑) |
