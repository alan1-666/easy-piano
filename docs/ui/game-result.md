> 完成一首曲目后的结算界面  
> 版本: v1.0.0 | 更新日期: 2026-04-18

## 1. 页面概述

游戏结束后展示评分结果、统计数据和奖励信息。通过动画揭晓星级和分数，增强成就感。页面从底部滑入。

## 3. 动画序列

```
时间轴:
  0ms      — 页面从底部滑入 (300ms)
  300ms    — "演奏完成!" 淡入
  500ms    — 歌曲名淡入
  800ms    — 第1颗星揭晓 (scale 0→1.3→1.0, 400ms spring)
  1200ms   — 第2颗星揭晓
  1600ms   — 第3颗星揭晓
  2000ms   — 分数滚动 (0→最终分数, 800ms)
  2800ms   — 评级字母弹入 (scale 2.0→1.0, 200ms)
  3000ms   — 统计数据逐个淡入 (每项100ms间隔)
  3400ms   — 最大连击淡入
  3600ms   — XP 奖励弹入 + 经验条动画填充 (600ms)
  4200ms   — 新成就卡片滑入 (如有)
  4500ms   — 操作按钮淡入
```

## 4. 评级标准

| 评级 | 条件 | 颜色 | 字体效果 |
|------|------|------|---------|
| S | 分数 ≥ 95% 最大分 | #FFD700 | 金色 + 闪光粒子 |
| A | 分数 ≥ 80% | #4A90D9 | 蓝色 |
| B | 分数 ≥ 60% | #50C878 | 绿色 |
| C | 分数 < 60% | #A0A0B0 | 灰色 |

## 7. 操作按钮行为

| 按钮 | 行为 |
|------|------|
| 重试 | 重新开始同一首曲目，进入倒计时 |
| 📤 分享 | 生成截图 → 调用系统分享面板 |
| 下一曲 | 进入下一首推荐曲目 (课程模式: 下一课) |
| 返回曲库 | 返回曲库 Tab 或课程列表 |

---

## 8. 边界情况

| 场景 | 处理 |
|------|------|
| 0 分 (全部 Miss) | 不显示星星，评级 C，鼓励文案 "再试一次，你可以的！" |
| 满分 (全 Perfect) | 彩色粒子特效，文案 "完美演奏！不可思议！" |
| 网络离线 | 分数本地保存，下次联网时同步；分享按钮不可用 |
| 升级 | XP 条填满时播放升级动画，弹出 "恭喜升级到 Lv.XX！" |
| 课程模式结算 | 额外显示 "课程进度: 3/10"，下一曲变为 "继续课程" |
| 无下一曲 | "下一曲" 按钮变为 "探索更多"，跳转曲库 |
| 新纪录 | 分数上方显示 "NEW BEST!" 金色标签 |
