# 首页 (Tab)

> App 主页面，用户每次打开 App 看到的第一个页面  
> 版本: v1.0.0 | 更新日期: 2026-04-18

---

## 1. 页面概述

首页是 App 的核心入口，汇集用户当前学习状态、练习目标、快速入口等信息，驱动用户每日练习。

## 8. 下拉刷新

| 操作 | 行为 |
|------|------|
| 下拉 | 触发刷新动画 (弹性下拉 → 加载指示器旋转) |
| 刷新内容 | 同步最新课程进度、练习记录、MIDI 状态 |
| 完成 | 弹回，内容更新 |
| 无网络 | 显示离线 Toast，使用本地缓存数据 |

---

## 9. 边界情况

| 场景 | 处理 |
|------|------|
| 新用户 (无任何数据) | 隐藏统计图，显示引导卡片 "开始你的第一次练习" |
| 连续打卡中断 | 火焰变灰色，显示 "重新开始连续打卡" |
| 今日目标已完成 | 进度条变绿色，显示 "🎉 今日目标已达成！" |
| 网络离线 | 顶部显示离线横幅，数据使用本地缓存 |
| 无 MIDI 设备 | MIDI 状态栏显示引导文案，可进入虚拟键盘模式 |
| 未登录/游客 | 问候语显示 "你好，琴友"，限制部分功能 |
