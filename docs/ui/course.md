## 6. 交互说明

### 6.1 课程列表

| 操作 | 行为 |
|------|------|
| 点击课程卡片 | 展开/收起课时列表 (200ms 展开动画) |
| 点击已完成课时 | 进入课时详情 (可回顾) |
| 点击当前课时 | 进入课时详情 |
| 点击未解锁课时 | Toast 提示 "请先完成前面的课程" |
| 点击锁定 Level | Toast 提示 "完成 Level X 后解锁" |
| 下拉刷新 | 同步最新进度 |

### 6.2 课时详情

| 操作 | 行为 |
|------|------|
| 完成知识讲解 | 解锁示范演奏步骤 |
| 观看完示范 | 解锁跟弹练习步骤 |
| 完成跟弹练习 | 解锁挑战关卡步骤 |
| 通过挑战 (≥1星) | 该课标记完成，解锁下一课 |
| 返回 | 返回课程列表 |

## 7. 边界情况

| 场景 | 处理 |
|------|------|
| 订阅过期 | Level 2+ 显示锁定 + "订阅解锁" 按钮 |
| 网络离线 | 已下载课程可离线学习，未下载的显示 "需要网络" |
| 课程更新 | 新课程推送提示 "有新课程可用" 红点 |
| 中途退出课时 | 保存进度，下次进入恢复到当前步骤 |
| Boss 关卡失败 | 可无限重试，显示鼓励文案 |
