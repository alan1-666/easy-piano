## 3. 交互说明

### 3.1 页面切换

| 操作 | 行为 |
|------|------|
| 左滑 | 进入下一页，动画: 平移 + 弹性 (300ms) |
| 右滑 | 返回上一页 |
| 点击"跳过" | 直接跳到最后一页 (或进入首页) |
| 点击圆点 | 跳转到对应页面 |
| 点击"开始探索" | 进入 MIDI 连接页或首页 |

### 3.2 自动播放 (可选)

| 项目 | 说明 |
|------|------|
| 自动切换 | 每页停留 4 秒后自动切换到下一页 |
| 用户触摸 | 用户触摸屏幕后取消自动切换 |
| 最后一页 | 不自动跳转，等待用户操作 |

### 3.3 首次引导后续

```
用户点击"开始探索"
    │
    ├── 已登录 (Apple ID 自动) ──→ 首页
    │
    └── 未登录 ──→ 登录注册页 (可选跳过)
                      │
                      ├── 完成登录 ──→ MIDI 连接引导 ──→ 首页
                      │
                      └── 跳过登录 ──→ MIDI 连接引导 ──→ 首页 (游客模式)
```

## 4. 状态与边界情况

| 场景 | 处理方式 |
|------|---------|
| 非首次启动 | 跳过引导，直接显示启动页→首页 |
| 引导中杀进程 | 下次启动从头开始引导 |
| 旋转设备 | 引导页支持竖屏，iPad 额外支持横屏 |
| 无网络 | 引导页不需要网络，可离线展示 |
| VoiceOver | 所有图片提供 accessibilityLabel，支持无障碍 |
