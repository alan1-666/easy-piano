# 曲库 Tab

> 曲目浏览、搜索与选择  
> 版本: v1.0.0 | 更新日期: 2026-04-18

---

## 1. 页面概述

曲库 Tab 提供所有可弹奏曲目的浏览、搜索和筛选功能。用户可以在此发现新曲目或选择想要练习的曲目。

## 7. 搜索交互

| 操作 | 行为 |
|------|------|
| 点击搜索框 | 键盘弹出 + 搜索框获焦 + 显示 "取消" 按钮 |
| 输入文字 | 实时搜索 (300ms 防抖)，结果替换列表 |
| 无结果 | 显示空状态: "未找到相关曲目" + 插画 |
| 点击取消 | 键盘收起 + 恢复原列表 |
| 搜索历史 | 搜索框下方显示最近 5 条搜索历史 |

## 8. 排序逻辑

| 排序选项 | 规则 |
|---------|------|
| 推荐 (默认) | 根据用户水平推荐合适难度的曲目 |
| 难度 ↑ | 按难度从低到高排序 |
| 难度 ↓ | 按难度从高到低排序 |
| 最新 | 按上架时间倒序 |
| 热门 | 按弹奏次数倒序 |

---

## 9. 边界情况

| 场景 | 处理 |
|------|------|
| 曲库为空 | 显示 "曲库加载中..." 或 "暂无曲目" |
| 网络离线 | 显示已缓存曲目，标注 "离线可用" |
| 大量曲目 | 分页加载 (每页 20 首)，滚动到底自动加载 |
| 购买成功 | Toast "购买成功！" + 状态标签变为 "已购买" |
| 订阅过期 | 已购买的单曲仍可用，订阅曲目显示锁定 |
| 无 MIDI 设备 | 仍可浏览曲库，点击 "开始练习" 提示连接设备 |
